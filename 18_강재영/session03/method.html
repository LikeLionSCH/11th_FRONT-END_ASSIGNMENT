String length : 문자열의 길이 반환 indexOf : 배역에서 지정된 값의 인덱스 반환 includes : 문자열에 지정된 값이 있는지 확인 split : 문자열을 배열로 분할 substring : 지정된 두 인덱스 사이의 문자를 추출 substr : 작성한 두 수에 해당하는 문자열과 문자열 사이의 문자열을 추출 trim : 문자열의 양 끝의 공백을 제거 toLowerCase : 문자열 소문자로 변환 toUpperCase : 문자열 대문자로 변환 replace : 지정된 값을 다른 값으러 변환 Number number : 숫자인지 다른 형태인지 판단하고 숫자면 숫자를, 다른 것이면 NaN을 반환 parselnt : 내가 지정한 진수 값에 따라 입력한 수를 변환(기본은 10진수) Math max : 가장 높은 값 반환 min : 가장 낮은 값 반환 floor : 해당 수를 가장 가까운 정수로 내림 변환 round : 해당 수를 가장 가까운 정수로 반올림 변환 random : 0에서 1사이의 난수 생성 sqrt : 해당 값에 루트를 씌우고 반환
